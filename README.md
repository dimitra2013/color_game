# color_game

This project is a colour guessing game created with Python. It uses the tkinter library to create the graphical user interface, including labels, an input box, colours, and the main game window. It also uses the random library to shuffle the list of colours and show a different word and text colour each time. The main idea of the game is that the user must type the colour of the word, not the actual word that is written. For example, if the word says “Red” but it is shown in blue, the correct answer is “Blue”. The game starts when the user presses Enter, then a 30-second timer begins. Each correct answer increases the score, the input box is cleared, and a new colour challenge appears. The app displays the instructions, the score, the remaining time, and the colour word. The user should type the correct colour in the text box and press Enter repeatedly until the time runs out.
